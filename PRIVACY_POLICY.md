# Hellfire Privacy Policy

Effective: 2025

Hellfire collects and stores **no personal user data**.

## What Hellfire Stores
Hellfire stores only:
- server ID
- configured member role
- cleanup settings
- welcome channel ID

This information is used only to operate the bot.

## Data Hellfire Does NOT Collect
Hellfire does NOT collect:
- usernames
- messages
- avatars
- behavioral data
- activity timestamps
- IP addresses
- device information

## Data Sharing
No data is shared with any third party.

## Data Removal
Removing the bot from a server permanently deletes that server's configuration.

## Contact
For issues or concerns, contact the bot owner.
