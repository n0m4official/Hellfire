# Hellfire — Precision Server Cleanup Bot

Hellfire is a lightweight, multi-server compatible cleanup bot designed to:
- remove inactive users
- purge no-role accounts
- remove no-avatar accounts
- automatically clean servers on a schedule
- detect raid activity
- welcome new members

Hellfire stores *no personal data*. Only server configuration is saved.

## Features
✔ Slash commands  
✔ Multi-server support  
✔ No personal data storage  
✔ Anti-raid detection  
✔ Scheduled cleanup  
✔ Welcome messages  
✔ Preview mode  
✔ Ultra-fast purges  

## Setup
1. Clone the repo
2. Install dependencies: 
```
pip install nextcord python-dotenv
```
3. Create a `.env` file:
```
DISCORD_TOKEN=YOUR_TOKEN_HERE
```
4. Run:
```
python main.py
```
